Title: {{class.instance}}->register($module, $dir)

----

Excerpt: Registers an additional file directory as source for a specific module

----

Params: -
  name: $module
  type: string
  text: Module for which the directory should be registered
-
  name: $dir
  type: string
  text: Path to file directory that should be registered

----

Return:

type: bool
text:

----

Text:

## Example

```php
kirby()->modules()->register('snippets',  __DIR__ . DS . 'snippets');
```
