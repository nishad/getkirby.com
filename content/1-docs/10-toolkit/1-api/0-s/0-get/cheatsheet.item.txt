Title: s::get

----

Excerpt: Gets a session value by key

----

Call: s::get($key = false, $default = null)

----

Return: type: mixed
text:

----

Params: - 
  name: $key
  type: mixed
  text: >
    The key to look for. Pass false or null
    to return the entire session array.
- 
  name: $default
  type: mixed
  text: >
    Optional default value, which should be
    returned if no element has been found
