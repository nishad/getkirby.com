Title: a::last

----

Excerpt: Returns the last element of an array

----

Call: a::last($array)

----

Return: type: mixed
text: The last element

----

Params: -
  name: $array
  type: array
  text: The source array

----

Text:

## Example

```php
<?php
$array = array(
  'cat',
  'dog',
  'bird',
);
$last = a::last($array);
var_dump($last);
```

Result:

```text
'bird'
```
