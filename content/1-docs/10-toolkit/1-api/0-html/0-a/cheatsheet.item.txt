Title: html::a

----

Excerpt: Generates an a tag

----

Call: html::a($href, $text = null, $attr = array())

----

Return: type: string
text: the generated html

----

Params: - 
  name: $href
  type: string
  text: The url for the a tag
- 
  name: $text
  type: mixed
  text: >
    The optional text. If null, the url will
    be used as text
- 
  name: $attr
  type: array
  text: Additional attributes for the tag
