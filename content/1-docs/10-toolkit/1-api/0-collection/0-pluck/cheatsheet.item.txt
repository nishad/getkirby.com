Title: {{class.instance}}->pluck($field, $split = null, $unique = false)

----

Excerpt: Extracts all values for a single field into a new array

----

Return:
type: array
text:

----

Params: -
  name: $field
  type: string
  text: The name of the field
-
  name: $split
  type: string
  text: The separator to split by
-
  name: $unique
  type: boolean
  text: If set to true, only unique values will be fetched, default is false.
