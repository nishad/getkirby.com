Title: Image

----
Since: 2.3.0

----

Excerpt:

A drag and drop image select field with preview thumbnail

----

Text:

The image field is a select field into which you can also drag and drop an image from the sidebar. It will show the filename and an image thumbnail as preview. The filename is saved in the content file.

## Example

```
fields:
  coverImage:
    label: Cover Image
    type: image
```
