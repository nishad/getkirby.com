Title:

{{class.instance}}->uri([$lang = null])

----

Excerpt:

Returns the URI. The URI is the relative URL without the host or any subfolders.

----
Params:-
  name: $lang
  type: string
  text: Optional, in case of a multi-language site; returns the translated URI

----
Return:

type: string
text:

----

Text:

## Example

```php
<?php echo $page->uri() ?>
```
