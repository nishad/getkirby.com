Title: {{class.instance}}->code()

----

Excerpt:

Returns a $files collection with all code files (css, js, html, etc.) in the page's directory

----

Return:

type: $files
text:

----

Text:

## Example

```php
<?php foreach($page->code()->filterBy('extension', 'css') as $css): ?>
<?php echo css($css->url()) ?>
<?php endforeach ?>
```
