Title:

{{class.instance}}->hide()

----

Excerpt:

Removes the sorting number from the page directory to switch it to invisible

----

Return:

type: boolean
text:

----

Text:

## Example

```php
try {

  page('visible-page')->hide();
  echo 'The page is now invisible';

} catch(Exception $e) {

  echo $e->getMessage();

}

```
