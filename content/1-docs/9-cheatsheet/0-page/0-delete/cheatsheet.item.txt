Title:

{{class.instance}}->delete($force = false)

----

Excerpt:

Deletes the $page object and all its data and files. This cannot be undone!

----

Params: -
  name: $force
  type: boolean
  text: If set to `true` the page is deleted even if there are subpages

----

Return:

type: boolean
text:

----

Text:

## Example

```php
try {

  page('blog/old-article')->delete();
  echo 'The page has been deleted';

} catch(Exception $e) {

  echo $e->getMessage();

}

```
