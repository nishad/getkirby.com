Title:

$files->flip()

----

Excerpt:

Returns the collection in reverse order

----

Return:

type: $files
text:

----

Text:

## Example

```php
<ul>
  <?php foreach($page->files()->flip() as $file): ?>
  <li>
    <a href="<?php echo $file->url() ?>">
      <?php echo html($file->filename()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
```
