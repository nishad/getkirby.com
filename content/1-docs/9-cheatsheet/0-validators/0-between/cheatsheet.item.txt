Title: Between

----

Excerpt: Checks for numbers within a given range

----

Return:

type: boolean
text:

----

Text:

## In your code

```php
if(v::between($input, 5, 20)) {
  echo 'Yay, valid!';
}

```

## In panel fields

```
fields:
  example:
    label: Example field
    type: text
    validate:
      between:
        - 5
        - 20
```
