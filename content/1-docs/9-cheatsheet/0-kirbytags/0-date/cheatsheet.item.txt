Title:

date

----

Excerpt: Adds a dynamic date to the text

----

Text:

## Examples and options:

The current year

```
(\date: year)
```

Any date defined by a PHP date format string

```
(\date: d.m.Y H:i:s)
```

You can read more about all (link: http://php.net/manual/en/function.date.php text: available formatting options on php.net)
