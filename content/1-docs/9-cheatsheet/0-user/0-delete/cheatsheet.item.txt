Title:

{{class.instance}}->delete()

----

Excerpt:

Deletes the user account. This cannot be undone

----

Return:

type: boolean
text:

----

Text:

## Example

```php
try {

  $site->user('bastian')->delete();
  echo 'The user has been deleted';

} catch(Exception $e) {

  echo 'The user could not be deleted';
  // optional reason: echo $e->getMessage();

}

```
