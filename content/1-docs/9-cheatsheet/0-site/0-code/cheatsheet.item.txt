Title:

{{class.instance}}->code()

----

Excerpt:

Returns a $files collection with all code files (css, js, html) placed directly in /content

----

Return:

type: $files
text:

----

Text:

## Example

```php
<?php foreach($site->code()->filterBy('extension', 'css') as $css): ?>
<?php echo css($css->url()) ?>
<?php endforeach ?>
```
