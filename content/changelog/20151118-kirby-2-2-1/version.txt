Text:

This bugfix release fixes all known bugs that unfortunately slipped into the 2.2.0 release. This release replaces 2.2.0. All users who have issues with the initial release should updated to fix those issues.

## Bugfixes

- The Panel installation bug has been fixed.
- A possible issue with the `$page->date()` method has been fixed.
- Fixed a small bug in the new datetime field
- A bug with translatable labels and help text for panel fields is fixed.

## Small additions

- New f::$units variable makes it possible to translate file size units used in f::niceSize() and all depending methods.
- Updated pt_PT translation
